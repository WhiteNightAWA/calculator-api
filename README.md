# Calculator API for Python
By White Night
## Install
```shell
# Linux/macOS
python3 -m pip install -U white-calculator

# Windows
py -3 -m pip install -U white-calculator
```


# QuickStart
```python
>>> from calculator import Calculator
>>> c = Calculator()
>>> print(c.run("1+8(5^2)"))
201
>>> print(c.run("9Ans"))
1809
```
